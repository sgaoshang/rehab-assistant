# 提醒助手 - 安装指南

本指南将帮助您在 Android 设备上安装提醒助手。

## 📱 系统要求

- **Android**: 5.0 (Lollipop) 或更高版本
- **存储空间**: 至少 50 MB 可用空间
- **网络**: 需要网络连接以下载安装包

---

## 📥 下载应用

### 方式 1：GitHub Release（推荐）

1. 访问下载页面：
   ```
   https://github.com/sgaoshang/rehab-assistant/releases/tag/v1.0.0
   ```

2. 根据您的设备选择版本：

## 📲 安装步骤

### 通用安装方法

1. **下载 APK 文件**
   - 使用手机浏览器打开下载链接
   - 等待下载完成（文件约 3-4 MB）
   - 确认下载的文件大小正确

2. **允许安装未知来源应用**
   - 点击下载的 APK 文件
   - 如果提示"禁止安装"或"未知来源"
   - 点击"设置"或"详情"
   - 开启"允许来自此来源的应用"或"安装未知应用"
   - 返回继续安装

3. **安装应用**
   - 点击"安装"按钮
   - 等待安装完成
   - 点击"打开"启动应用

## 🛠️ 特定设备安装指南

### 小米手机（MIUI）

**MIUI 12 及以上：**
1. 下载 APK 文件
2. 设置 → 应用设置 → 应用管理
3. 右上角 ⋮ → 显示所有应用
4. 找到"浏览器"或"下载管理"
5. 安装未知应用 → 开启
6. 返回点击 APK 安装

**MIUI 10/11（推荐下载兼容版）：**
1. 设置 → 更多设置 → 系统安全
2. 特殊应用权限 → 安装未知应用
3. 找到"浏览器"或"文件管理"→ 开启
4. 或：设置 → 安全 → 纯净模式 → 选择"标准模式"
5. 返回点击 APK 安装

**如果提示"解析包错误"：**
- 使用**兼容版 APK**
- 删除之前下载的文件
- 清空浏览器缓存后重新下载

### 华为手机（EMUI/HarmonyOS）

1. 下载 APK 文件
2. 设置 → 安全 → 更多安全设置
3. 外部来源应用下载 → 开启
4. 或在安装时选择"仍要安装"
5. 完成安装

### OPPO/vivo 手机

1. 下载 APK 文件
2. 设置 → 安全 → 安装外部来源应用
3. 找到浏览器 → 开启
4. 返回点击 APK 安装

### 原生 Android / 其他品牌

1. 下载 APK 文件
2. 设置 → 安全
3. 未知来源 → 开启
4. 或在安装时选择"允许"

## ✅ 首次使用配置

### 1. 授予权限

首次启动应用时，需要授予以下权限：

- **通知权限**：用于定时提醒
  - 点击"允许"授予权限
  - 如误点"拒绝"，可在设置中重新开启

### 2. 添加第一个项目

1. 点击右下角"设置"标签
2. 点击"添加项目"
3. 选择预设项目或创建自定义项目
4. 设置提醒时间
5. 保存项目

### 3. 测试提醒

1. 添加一个近期时间的提醒（如 5 分钟后）
2. 锁定屏幕等待
3. 确认收到通知提醒

## ❓ 常见问题

### Q1: 下载的文件无法打开？

**A:** 确认文件下载完整
- 标准版应为 3.5 MB 左右
- 兼容版应为 3.2 MB 左右
- 文件过小说明下载不完整，请重新下载

### Q2: 提示"应用未安装"或"安装失败"？

**A:** 尝试以下方法：
1. 卸载旧版本（如果存在）
2. 清理设备存储空间（至少保留 100 MB）
3. 重启设备后重新安装
4. 尝试使用兼容版 APK

### Q3: 小米手机提示"解析包错误"？

**A:** 这是常见问题，请：
1. 下载**兼容版 APK**（专为 MIUI 优化）
2. 删除所有之前下载的 APK 文件
3. 清空浏览器缓存
4. 重新下载兼容版并安装

### Q4: 收不到通知提醒？

**A:** 检查以下设置：
1. 应用通知权限已开启
2. 系统通知未被关闭
3. 电池优化未限制应用
4. 允许应用后台运行

### Q5: 如何允许应用后台运行？

**A:** 不同品牌设置不同：
- **小米**：设置 → 应用设置 → 应用管理 → 提醒助手 → 省电策略 → 无限制
- **华为**：设置 → 应用 → 应用启动管理 → 提醒助手 → 手动管理 → 全部开启
- **OPPO**：设置 → 电池 → 应用耗电管理 → 提醒助手 → 允许后台运行
- **vivo**：设置 → 电池 → 后台耗电管理 → 提醒助手 → 允许后台运行
