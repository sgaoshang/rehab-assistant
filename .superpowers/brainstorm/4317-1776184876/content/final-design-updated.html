交互流程示例

场景1：用户使用模板
→ 页面打开，已选时间为空，下拉框显示"选择常用模板（可选）"
→ 选择"每日三次 - 早中晚"
→ 时间自动填充为 08:00, 12:00, 18:00
→ 点击"保存项目"完成
场景2：用户切换模板
→ 选择"每日三次 - 早中晚"后有 08:00, 12:00, 18:00
→ 切换到"每日三次 - 饭前"
→ 时间变为 07:30, 08:00, 11:30, 12:00, 17:30, 18:00（追加去重）
→ 用户手动删除不需要的时间
场景3：用户只用自定义时间
→ 页面打开，已选时间为空
→ 直接点击"+ 添加自定义时间"
→ 弹出模态框，选择时间（例如 09:15）
→ 确认后，时间变为 09:15
→ 继续添加更多自定义时间
